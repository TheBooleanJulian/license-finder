## What it does

License Finder is a single-file HTML tool that helps developers quickly identify and choose the right open-source license for their projects. Instead of trawling through lengthy legal documents, you get a straightforward interface to compare and find the license that fits your needs.

## Features

- Browse and compare common open-source licenses
- Single-file, zero-dependency — just open `index.html`

## Tech Stack

| Layer | Choice |
|---|---|
| Frontend | Single-file HTML |

## Quick Start

No install needed. Clone the repo and open the file:

```bash
git clone https://github.com/TheBooleanJulian/license-finder
cd license-finder
open index.html
```
