## What it does

License Finder is a single-file HTML tool that helps developers quickly identify and choose the right open-source license for their projects. Instead of trawling through lengthy legal documents, you answer a short quiz and get a direct recommendation — including dual-licensing options like AGPL + commercial for projects that need them.

## Features

- Quiz-driven license recommendation — answer a few questions, get a result
- Includes AGPL + commercial dual-licensing as a quiz option
- Per-license template folders linked directly from the tool
- Single-file, zero-dependency — just open `index.html`

## Quick Start

No install needed. Clone the repo and open the file:

```bash
git clone https://github.com/TheBooleanJulian/license-finder
cd license-finder
open index.html
```
